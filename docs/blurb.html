Still going strong in it's 6th year as an open, online course/community, the classic ed-tech groove of DS106 included a distributed network of participants united with RSS syndication, an open assignment bank populated with response from the syndication, and a daily show of creative activities that will on its 1975th episode by the time this conference rolls around.

Since then I and others have replicated the Feed Wordpress syndication model in more class halls than we can count. Both the DS106 Daily Create and the DS106 Assignment Bank have been re-crafted into configurable Wordpress themes so anyone can create sites with the same functionality.

Learn about how these cover sites have done in different contexts and how you can set one up yourself. We'll give people a hands on experience rolling out one of these quickly within Reclaim Hosting's State U demo environment.
